⚙️ Configuração

Conecte esta página ao seu Google Sheets em 3 passos

📋 Como funciona

Você alimenta os dados na planilha Google Sheets → esta página lê automaticamente a cada 60 segundos → tudo atualiza sem você precisar mexer aqui.

Passo a passo para conectar

1
Importe a planilha para o Google Sheets
Baixe o arquivo mversity_sheets_modelo.xlsx → abra o Google Drive → arraste o arquivo → clique com o botão direito → "Abrir com" → "Planilhas Google"
2
Publique a aba "Tarefas" na web como CSV
No Google Sheets: Arquivo → Compartilhar → Publicar na web
Selecione a aba "Tarefas" e o formato "Valores separados por vírgula (.csv)"
Clique em "Publicar" e copie o link gerado
3
Publique a aba "Semanas" na web como CSV
Repita o processo acima mas selecione a aba "Semanas"
Conectando...

⚠️ Dica importante

O link do Google Sheets precisa ser público para esta página conseguir ler.
Não se preocupe: o link só permite leitura — ninguém consegue editar sua planilha por ele.
Para revogar o acesso a qualquer momento: Arquivo → Compartilhar → Publicar na web → Parar publicação.
